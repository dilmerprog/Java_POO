# 📚 POO_Java

Projeto desenvolvido para praticar conceitos de *Programação Orientada a Objetos (POO)* utilizando a linguagem Java.

---

## 🚀 Objetivo

Este repositório tem como foco o aprendizado e aplicação dos principais conceitos de POO:

* ✅ Classes e Objetos
* ✅ Métodos
* ✅ Encapsulamento
* ✅ Herança
* ✅ Polimorfismo

---

## 🛠️ Tecnologias utilizadas

* ☕ Java
* 💻 IntelliJ IDEA

---

## 📂 Estrutura do projeto


```bash
POO_Java/
 ├── src/
 │    └── Main.java
 ├── .gitignore
 └── README.md
```

---

## ▶️ Como executar

1. Clone o repositório:


git clone https://github.com/dilmerprog/POO_Java.git


2. Abra o projeto no IntelliJ

3. Execute a classe Main.java
